5. 轻商城

5 litemall轻商城

litemall轻商城，是商城移动版本。

技术：

  • 轻商城前端，即litemall-vue模块
    • vue-cli3
    • Vue + Vue-router + Vant + Sass
    • axios
    • fastclick
    • babel-polyfill
    • @xkeshi/vue-countdown
    • Vant
  • 轻商城后端，即litemall-wx-api模块，也就是和小商城后端是一样的。
    • Spring Boot 2.x
    • Spring MVC
    • weixin-java-tools
